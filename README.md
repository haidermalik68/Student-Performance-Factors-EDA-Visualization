# Student Performance Factors — Data Analysis & Visualization

## Overview
Exploratory data analysis (EDA) on a real world student performance 
dataset to uncover key factors influencing exam scores.

## Dataset
- Source: Kaggle — Student Performance Factors
- Features: Study Hours, Attendance, Sleep, Tutoring, 
  Parental Education, Gender and more
- Target: Exam Score

## Libraries Used
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn

## Analysis Performed
- Impact of study hours on exam scores
- Attendance vs exam score relationship
- Sleep hours effect on performance
- Gender wise performance comparison
- Parental education influence on scores
- Correlation heatmap of all variables
- Subplots comparing multiple factors

## Key Findings
- Study hours and attendance most strongly impact scores
- Sleep below 6 hours negatively affects performance
- Tutoring sessions significantly improve results
- Higher parental education = better student performance

## How to Run
1. Clone this repository
2. Install required libraries
pip install pandas numpy matplotlib seaborn
3. Open StudentPerformance.ipynb in Jupyter or VS Code
4. Run all cells
